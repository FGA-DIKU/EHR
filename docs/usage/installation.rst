Installation
===========

Requirements
-----------
* Python 3.8+
* PyTorch 1.8+
* Transformers 4.0+

Installation Steps
----------------
1. Install from PyPI:
   
   .. code-block:: bash

      pip install corebehrt

2. Install from source:

   .. code-block:: bash

      git clone https://github.com/yourusername/corebehrt.git
      cd corebehrt
      pip install -e .
