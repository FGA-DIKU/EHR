# EHR

## Unittests
### In Linux
Create a python virtual environment and install the requirements:
```
$ python -m venv .venv
$ source .venv/bin/activate
(.venv) $ pip install -r requirements.txt
```
Run the unittests:
```
(.venv) $ python -m unittest
```

## Mandatory naming
* Date of death: 'DEATHDATE'
* Date of birth: 'BIRTHDATE'
